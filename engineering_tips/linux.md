# Linux Commands
```bash
history | grep uvicorn # find all commands containing "uvicorn"
lspci # list all PCI devices
df -h # display disk space usage
du -sh # display disk usage for the current directory
grep MemoTotal /proc/meminfo # get total memory
cat /proc/cpuinfo # get CPU information
sudo ss -tulpn | grep :11560 # find all connections to port 11560
```
